## 1. 初次启动

双击运行程序后，小宠物就会出现在屏幕右下角附近，同时系统托盘（右下角任务栏小图标区）会多一个小宠物图标。

首次启动会在你的用户目录下自动创建配置文件夹 `~/.desktop_pet/`（Windows 上一般在 `C:\Users\你的用户名\.desktop_pet\`），里面存着所有设置，不用手动创建。

## 2. 和小宠物玩耍

### 拖拽
左键按住小宠物，拖到任意位置松手——它会根据位置掉落到地面、贴墙、或挂到天花板上。

### 戳一下（单击）
左键点一下小宠物（不要拖动），它会播放一个戳的小动画：
- 在地面上戳 → 地面反应动画
- 在天花板上戳 → 掉下来
- 在墙上戳 → 墙上反应动画
- **睡着时戳 → 叫醒它！**

### 摸头
左键**按住不动**约半秒（不要拖），会触发摸头动画，小宠物会很开心。

### 缩放大小
把鼠标放在小宠物上，按住 **Ctrl + 滚轮** 就能放大或缩小。  
`Ctrl+=` 放大 / `Ctrl+-` 缩小 / `Ctrl+0` 恢复默认大小。

### 右键菜单
右键点击小宠物，弹出快捷菜单：
- **复位** —— 把宠物拉回屏幕中央底部
- **安静模式** —— 暂停所有气泡和提醒（拖拽、戳、摸头照常）
- **活动气泡** —— 开关应用检测气泡
- **强制睡觉** —— 立刻让宠物躺下
- **桌宠控制台** —— 打开 AI 对话窗口
- **设置** —— 打开完整设置面板
- **退出** —— 彻底关闭桌宠

### 系统托盘图标
右下角任务栏的小宠物图标也有类似菜单，功能一样。

## 3. 小宠物的日常状态

小宠物会在这些状态之间自然切换：

| 状态 | 说明 |
|------|------|
| **站着发呆** | 默认状态，原地播放 idle 动画 |
| **散步** | 随机左右走动（可在设置里关闭自动走路） |
| **下落** | 从高处掉下来（拖到空中松手、从天花板/墙壁掉落时） |
| **贴墙滑落** | 走到屏幕边缘时贴墙，会缓缓滑下来 |
| **挂天花板** | 拖到屏幕顶部松手，会挂一会儿再掉下来 |
| **睡觉** | 长时间不动会自动入睡（详见下一节） |

如果你安装了多个 idle 动画（`idle.gif` + `idle2.gif`），宠物会自动在两个动画之间切换，看起来更生动。

## 4. 睡觉与起床

### 什么时候会睡着？
- 小宠物在地面上**一直没人理**超过一定时间（默认 20 分钟），就会自动进入睡眠。
- 刚启动后有一段**兴奋期**（默认 10 分钟），兴奋期内不会睡着。

### 白天 vs 夜晚
- **7:00 ~ 19:00** 白天：播放 `sleep_day.gif`
- **19:00 ~ 次日 7:00** 夜晚：播放 `sleep_night.gif`
- 如果你只准备了一个 `sleep.gif`，它会全天候使用

### 怎么叫醒？
**戳一下**就醒了！醒来后有一段保护时间（默认 45 秒），不会立刻又睡回去。

### 在设置里调整
- 打开设置 → 桌宠设置 → 睡眠
- 可以调整入睡时间、兴奋期、或者直接关闭自动睡觉

## 5. 生活提醒（喝水 / 动一动）

小宠物会定时提醒你喝水和动一动，像一个贴心小伙伴。

- **喝水提醒**：默认每 30 分钟一次
- **动一动提醒**：默认每 45 分钟一次
- **生效时段**：默认 9:00 ~ 23:30（深夜不打扰你）

提醒方式：小宠物会播放特殊动画（如果有 `life_water.gif` / `life_move.gif`），或者弹一个小气泡。

如果宠物正在睡觉，提醒会**排队等它醒来再显示**（醒来时队列会清空，避免一下子弹好几条）。

设置路径：设置 → 桌宠设置 → 提醒

## 6. 应用气泡（切应用时冒话）

这是一个很好玩的功能：当你切换前台应用时，小宠物可能会冒出一句话，比如你打开微信时说"又去摸鱼啦？"

### 工作原理
1. 小宠物每 300 毫秒检测一次你当前在用什么应用
2. 根据应用映射表，识别出应用名和分类（如"聊天"、"办公"）
3. 从文案池里挑一句话，以气泡形式弹出

### 可以调的参数
- **触发概率**：默认 60%，不是每次切应用都会说话
- **冷却时间**：同一分类的应用，说完一次要等一段时间才会再说
- **仅切换时触发**：打开后，只有换到不同应用时才触发，同一个应用内不会反复说
- **显示时长**：气泡在屏幕上停多久

设置路径：设置 → 应用气泡

## 7. 文案池（自定义气泡文案）

你可以给每个应用、每个分类定制宠物说的话：

### 三个层级
1. **专属文案**：给某个具体应用写的话（比如给"微信"写"又聊天去了～"）
2. **分类文案**：给一个分类写的话（比如给所有"聊天"类应用写"社交达人！"）
3. **通用文案**：所有分类共享的兜底文案

还可以设置"**混合模式**"：让专属文案和分类通用文案一起随机出现，更丰富。

设置路径：设置 → 应用气泡 → 文案池

## 8. AI 对话（桌宠控制台）

要和小宠物聊天，你需要先配置 AI 服务。

### 怎么打开？
- 右键小宠物 → 桌宠控制台
- 或者托盘图标右键 → 桌宠控制台

### 怎么配置 AI？
打开设置 → AI 功能 → 连接配置：
1. **选服务商**：支持 OpenAI、DeepSeek、通义千问、月之暗面、Anthropic 等十多个
2. **填 API Key**：从对应服务商的官网获取
3. **选模型**：可以手动输入，也可以点"获取模型"自动拉取列表
4. **测试连接**：点一下验证配置是否正确

### 对话功能
- 输入框打字，按 **Enter** 发送
- **Shift+Enter** 换行
- 勾选"**附带截图**"可以把当前屏幕画面一起发给 AI（需要支持视觉的模型）
- AI 回复会以气泡形式弹在桌面上

### 人设预设
你可以给宠物设定不同的性格，比如：
- 傲娇猫娘
- 正经助手
- 毒舌损友

也可以自己写自定义人设。选了预设再切回"（自定义）"时，输入框会清空（或恢复你之前自己写的内容），不会残留预设。

### 记忆黑匣子
控制台里点"记忆黑匣子"可以看到所有历史对话，支持：
- **重试** —— 用相同问题重新问一次
- **编辑** —— 修改问题后重新发送
- **删除** —— 删掉某条记录
- **清空** —— 一键清除所有历史

## 9. AI 自动巡视

开启后，小宠物会每隔一段时间自动截屏，然后基于画面内容主动发言。就像宠物在"看"你在做什么，然后说点应景的话。

- 在控制台里勾选"自动巡视"，或者在设置 → AI 功能里配置间隔时间
- 间隔设为 0 就是关闭
- 巡视结果会存进记忆黑匣子

## 10. 设置面板总览

打开方式：右键小宠物 → 设置

### 三个主标签页

**桌宠设置**
| 子页 | 内容 |
|------|------|
| 行为 | 移动速度、思考间隔、自动走路、游走半径、边距、自动掉落 |
| 睡眠 | 睡眠开关、入睡时间、兴奋期、强制睡/醒 |
| 提醒 | 喝水/动一动间隔、生效时段、提示停留时间、待机闲聊间隔 |
| 安静与路径 | 安静模式开关、换皮提示、配置路径 |

**应用气泡**
| 子页 | 内容 |
|------|------|
| 全局参数 | 触发概率、显示时长、前台判定延迟 |
| 分类冷却 | 每个分类的冷却秒数 |
| 应用映射 | 管理已识别的应用（含自动抓取、手动添加、网站识别） |
| 文案池 | 专属/分类/通用文案管理 |
| 过滤器 | 屏蔽进程列表、屏蔽标题关键词 |

**AI 功能**
| 子页 | 内容 |
|------|------|
| 连接配置 | 服务商、Base URL、API Key、模型 |
| 人设与回复 | 系统 Prompt 预设、回复字数限制 |
| 高级 | 记忆轮数、自动截屏间隔、黑匣子容量 |

## 13. 配置文件在哪？

所有配置都存在 `~/.desktop_pet/` 文件夹下：

| 文件 | 作用 |
|------|------|
| `ai_settings.json` | AI 连接信息、模型、人设、回复长度等 |
| `pet_settings.json` | 行为参数（速度、睡眠、提醒等） |
| `app_map.json` | 应用识别映射表 |
| `bubbles.json` | 气泡设置和所有文案 |
| `filters.json` | 进程/标题屏蔽规则 |
| `chat_memory.json` | AI 对话历史（记忆黑匣子） |

一般不需要手动编辑这些文件，通过设置面板操作就行。如果想备份或迁移，复制整个文件夹即可。

## 14. 常见问题 Q&A

**Q：小宠物不动了，是不是卡了？**  
A：大概率是**睡着了**！戳一下就能叫醒。如果想关闭自动睡觉，去设置 → 桌宠设置 → 睡眠 → 取消勾选。

**Q：AI 对话没反应？**  
A：请检查：
1. 有没有填 API Key？（设置 → AI 功能 → 连接配置）
2. 点"测试连接"看看是否通？
3. 模型名是否正确？可以点"获取模型"自动拉取

**Q：怎么让宠物不那么话多？**  
A：几个思路：
- 降低触发概率（应用气泡设置里）
- 增加分类冷却时间
- 开启"仅切换时触发"
- 直接开安静模式（右键 → 安静模式）

**Q：我想让宠物在特定应用下说特定的话？**  
A：设置 → 应用气泡 → 文案池 → 选对应应用 → 添加专属文案。

**Q：打开设置面板后桌宠的标题栏是黑色的？**  
A：这已经修复了。如果你在用旧版本，更新到最新版即可。

**Q：怎么彻底退出？**  
A：关闭设置窗口**不会**退出桌宠；需要右键小宠物（或托盘图标）→ 退出。

**Q：怎么换宠物形象？**  
A：替换 `assets/` 文件夹里的 GIF 文件就行，文件名要对应（见第 11 节）。

**Q：提醒太烦了怎么办？**  
A：设置 → 桌宠设置 → 提醒 → 调大间隔时间，或者直接取消勾选。
